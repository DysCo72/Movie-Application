REST API Documentation

The REST API allows users to obtain information about the movies or categories for oscars Awards.

This pages list instructions for the use of the API.

Find Nomination by ID

http://localhost:8080/awardmovies/id/[id]

[id]: Valid ID will be integer with a value between 0 and 10394.

Find Nomination by year

http://localhost:8080/awardmovies/year/[year]

[year]: Valid year will be integer with a value between 1928 and 2020.

Find Nomination by Categories for rewards

http://localhost:8080/awardmovies/[categories]/[year]

[year]: Valid year will be integer with a value between 1928 and 2020.

[categories]:

  • animated-feature
  • costume-design
  • film-editing
